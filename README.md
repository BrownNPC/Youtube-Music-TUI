# Getting Started

Run ```ytt``` to launch the app.


###### You can type ``ytt -h`` for the help message

```
usage:	ytt [options]
options:
  -h, help, --help       Show this help message
  -c, config, --config   Open config file folder
  -r, refresh, --refresh Refresh the playlist cache
```
